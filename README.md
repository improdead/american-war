# American War — Desk Edition (ENG4U1-21 ISP Website)

**Student:** Dekai Li  
**Professor:** Haughton, Sheleah  
**Course:** ENG4U1-21  

Interactive scrapbook folio for Omar El Akkad's *American War*: summary, thesis, characters, feminist MLA essay, video, narration, and Works Cited.

## View the site

1. Open `index.html` in a browser (double-click locally), **or**
2. Enable **GitHub Pages** on this repo: Settings → Pages → Source: `main` branch, `/ (root)` → your URL will be  
   `https://improdead.github.io/american-war/`

Submit that live URL to your teacher if required.

## Record your audio (ISP requirement)

1. Open **`assets/narration-script.txt`** or the script on **page 20** of the book.
2. Read it **in your own voice** (~3½ minutes). Start with “Hi, I'm Dekai Li…” so the marker knows it's you.
3. Save as **`assets/narration.mp3`** and push to GitHub.
4. Test: open the site → page 20 → Play.

Works Cited entry (already on page 22):
